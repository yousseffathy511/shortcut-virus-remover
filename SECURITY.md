# Security Policy

## Supported Versions

The latest GitHub Release is the supported version of Shortcut Virus
Remover. Older releases may contain bugs that have already been fixed.

## Reporting A Security Issue

If you find a security problem in Shortcut Virus Remover, please report
it responsibly.

Contact:

- GitHub: [@yousseffathy511](https://github.com/yousseffathy511)
- Project: [shortcut-virus-remover](https://github.com/yousseffathy511/shortcut-virus-remover)

Please include:

- A clear description of the issue.
- Steps to reproduce it.
- The affected version or release tag.
- Any relevant logs from `%TEMP%\shortcut-virus-remover.log`.
- Whether the issue affects the Windows host cleanup, USB cleanup,
  Defender exclusions, or release executable.

## Scope

Security reports are welcome for:

- Unsafe deletion behavior.
- False positives that could delete normal user files.
- Missed malware persistence from the shortcut virus / Runner family.
- Problems with the GitHub Actions release build.
- Suspicious behavior in the compiled executable.

## Disclosure

Please do not publish exploit details publicly before the maintainer has
had a reasonable chance to investigate and fix the problem.
